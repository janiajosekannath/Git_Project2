# Linux & Command Line Basics Project 1

## Project Overview

This project was completed as part of the DecodeLabs DevOps Internship Program. The objective of the project is to gain hands-on experience with Linux command-line operations and understand the fundamentals of file and directory management in a Linux environment.

## Objectives

- Learn basic Linux commands.
- Understand Linux file system navigation.
- Create and manage directories and files.
- Perform file operations such as copying, renaming, and deleting.
- Develop familiarity with the Linux terminal environment.

## Commands Practiced

- pwd
- ls
- cd
- mkdir
- touch
- cat
- cp
- mv
- rm

## Tasks Performed

- Created project directories.
- Created and managed files.
- Navigated through directories.
- Viewed file contents.
- Copied and renamed files.
- Deleted files and verified directory structure.

## Tools Used

- Ubuntu (WSL)
- Linux Terminal

## Learning Outcomes

Through this project, I gained practical knowledge of Linux command-line operations and developed a strong foundation for working with Linux-based systems in DevOps environments.
